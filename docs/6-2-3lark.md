#### Lark (Feishu) Configuration

1. In a Lark Group, click Settings -> **Bots**.

![lark_1](images/image-20211220221635593.png)

2. **Add Bot** -> **Custom Bot**.

![lark_2](images/image-20211220220515428.png)

3. Copy the **Webhook URL**. HFish usually needs the token part.

![lark_3](images/image-20211220221136447.png)

4. Paste into HFish, Test, and Save.
   *(Add "HFish" as a keyword if required by security settings).*

![lark_4](images/image-20211220220958972.png)
