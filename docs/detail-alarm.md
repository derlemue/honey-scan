#### Alert Configuration

This page configures Alert Policies and Notification Channels.

**[Alert Policy]**: Define Rules (What to send).
**[Notification Config]**: Define Channels (Where to send).

![alert_main](images/20210812135455.png)

![alert_policy](images/20210812135501.png)

**Channels:**
- **Syslog**: Up to 5 servers. TCP/UDP.
- **Email**: SMTP.
- **Webhook**: DingTalk, WeChat Work, Lark, Custom.
