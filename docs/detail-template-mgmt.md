#### Template Management

Templates allow you to define a standard set of up to **10 honeypot services** that can be applied to multiple nodes.

> **Create Template**

Custom name, description, and select services.

![create_template](../images/20210616170818.png)

> **Adjust Protocol**

For Web, Network Device, Security Device, and IoT services, you can toggle between **HTTP** and **HTTPS** to better match your environment.

![adjust_protocol](../images/20210812135158.png)

> **Apply Template**

Go to **[Node Management]**, expand a node, and select the template.
*Any changes made to a template will automatically update all nodes using that template.*
